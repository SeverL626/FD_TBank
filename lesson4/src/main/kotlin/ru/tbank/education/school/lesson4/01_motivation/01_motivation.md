Дан следующий код:

```kotlin

    val user = database.findUser(userId)
    val email = user.email
    sendEmail(email, "Welcome!")
    println("Email sent successfully")
}
```

**Задание**:

- Перечислите все возможные ошибки, которые могут возникнуть в этом коде

- Предложите, как можно улучшить этот код

## Задача 3: Fail Fast vs Defensive Programming

Даны два подхода к написанию функции деления:

**Подход A:**

```kotlin
fun divide(a: Int, b: Int): Int {
    if (b == 0) throw IllegalArgumentException("Division by zero")
    return a / b
}
```

**Подход B:**

```kotlin
fun divide(a: Int, b: Int): Int {
    if (b == 0) return 0
    return a / b
}
```

**Задание**:

- Какой подход лучше и почему?
- В каких ситуациях можно использовать подход B?
- Приведите пример реальной функции, где каждый подход был бы уместен
