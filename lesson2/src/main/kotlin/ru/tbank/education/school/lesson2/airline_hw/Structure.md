## Person (люди)

### Clients (клиенты)

#### CommonClients (обычные клиенты)
- firstname (имя)
- surname (фамилия)
- age (возраст)
- sex (пол)
- passport_id (номер паспорт)
- registration_date (дата регистрации)
- birth_date (дата рождения)
- country_of_birth (страна рождения)
- adress (физический адрес регистрации)
- children (кол-во детей)
- spouse (наличие супруга/супруги)
- service_class (класс обслуживания)
- loayal_points (баллы лояльности)
- last_flight (дата последнего полета)
- first_flight (дата первого полета)
- number_of_flights (кол-во полетов)

#### VIPClient (VIP клиенты)
- firstname (имя)
- surname (фамилия)
- age (возраст)
- sex (пол)
- passport_id (паспорт)
- registration_date (дата регистрации)
- birth_date (дата рождения)
- country_of_birth (страна рождения)
- adress (адрес)
- children (кол-во детей)
- spouse (наличие супруга/супруги)
- vip_category (категория VIP)
- vip_reason (причина VIP)
- rest_room_number (номер бизнес-зала/комнаты в аэропортах)
- plane_type (тип самолета)
- personal_plane (ин-фа о частном (персональном) самолёте)

## Airplane (самолёты)
- plane_number (номер самолета)
- model (модель)
- manufacturer (производитель)
- seats (кол-во мест)
- production_year (год выпуска)
- range_km (максимальная дальность полета, км)
- max_takeoff_weight (максимальная взлетная масса, кг)
- engines (количество двигателей)
- engine_type (тип двигателя)
- wifi (наличие Wi-Fi: true/false)
- need_repair (требуется ли ремонт: true/false)

## Flights (рейсы, sealed class)

### SheduledFlights (регулярные рейсы)
- flight_international_code (международный код рейса)
- flight_number (внутренний номер рейса)
- departure (время вылета)
- arrival (время прилета)
- airplane (самолет)
- international (международный рейс: true/false)
- loyalty_points_coef (коэффициент баллов)
- demand (спрос)
- creation_date (дата создания)
- any_technical_problems (рейс отменён по причине: технические проблемы)
- any_passengers_problems (рейс отменён по причине: проблемы с пассажирами)
- any_staff_problems (рейс отменён по причине: проблемы с персоналом)
- any_weather_problems (рейс отменён по причине: погодные условия)
- any_authority_problems (рейс отменён по причине: запрет от Федерального Управление Гражданской Авиации)
- cancelled (отменен ли рейс: true/false)

### PersonalFlights (VIP / персональные рейсы)
- flight_international_code (международный код рейса)
- flight_number (внутренний номер рейса)
- departure (время вылета)
- arrival (время прилета)
- airplane (самолет)
- personal_pilot_1 / personal_pilot_2 (ФИО пилотов)
- personal_stewardess_1 / personal_stewardess_2 (ФИО стюардесс)
